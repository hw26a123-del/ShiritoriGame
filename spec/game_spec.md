# しりとりゲーム仕様書「爆速しりとり」

## 1. 概要
制限時間内にタイピングでしりとりを続ける、PC向けのゲーム。タイトルは「爆速しりとり」。

## 2. 画面レイアウト（UI構成）
ゲームプレイ中は以下のレイアウトで構成される：
- **画面上部中央**: 最初の単語、およびしりとりで繋がっている現在の単語を表示。
- **画面中部中央**: プレイヤーが文字を入力するテキスト入力欄。
- **画面上部右**: タイマー（残り時間数値）を表示。
- **画面上部左**: スコアを表示。
- **画面下部**: タイマーの残り時間を示す進捗バー（インジケーター）を表示。

## 3. ゲームの進行状態と画面表示

### 3.1 スタート画面
ゲーム開始前はスタート画面を表示する。
- **画面上部中央**: タイトル「爆速しりとり」を表示。
- **画面中央**: 以下の要素を上から順に表示。
  1. **スタートボタン**
  2. **最高スコア**

### 3.2 ゲームオーバー画面
ゲームオーバー時は以下の要素を上から順に画面中央に表示する：
1. **リスタートボタン**
2. **最終スコア**
3. **最高スコア**

## 4. ゲームルール（コアメカニクス）
- **入力操作**: PC（キーボード）によるタイピング入力。
- **ゲームの進行（システムとの交互プレイ）**:
  - ゲームは「プレイヤーのターン」と「システムのターン」が交互に繰り返される。
  - プレイヤーが正しい単語を入力して確定（Enter）すると、システムが自動的に次に繋がる単語を提示する。
  - プレイヤーは、システムが提示した最新の単語から繋がるように次の単語を入力する。
- **入力・確定ルール**:
  - ひらがなのみの入力を対象とする。
  - Enterキーが押されたときに入力を確定し、判定を行う。
  - 誤った入力（しりとりとして繋がっていない、すでに使われた、等）やひらがな以外の文字が入力された、あるいは辞書判定で存在しないとされた場合は、入力された文字が震えながら消去される。
- **ゲーム開始条件**:
  - ゲーム開始時の最初の単語は、システム側があらかじめ用意した「3文字の単語」を表示する。ただし、この単語の最後の文字に「ん」は使用しない。
- **辞書判定**:
  - 入力された単語について、無料かつ事前登録不要の外部辞書APIを利用して実在する言葉かどうかの判定を行う。
- **制限時間（タイマー）システム**:
  - ゲーム開始時、制限時間は30秒。
  - タイマーが0秒（残り時間なし）になった場合は、ゲームオーバーとなる。
  - しりとりを一回成功（完了）させるごとに、次回入力のタイマーの最大時間が1秒ずつ減少。
  - タイマー最大時間の最小値は5秒（5秒未満には減少させない）。
- **しりとり判定の特殊ルール**:
  - **濁音・半濁音の緩和（一方通行動作）**: 
    - 前の単語が濁音・半濁音で終わる場合、濁点・半濁点を取り除いた清音から始まる言葉で繋いでもよい。（例：「らくご」の「ご」から「こあら」の「こ」として繋ぐことは許容する）
    - 前の単語が清音で終わる場合、次の単語を濁音・半濁音で始めることはできない。（例：「らっこ」の「こ」から「ごりら」の「ご」に繋ぐことは不可）
  - **長音（ー）の処理**: 「るびー」のように最後が長音符「ー」で終わる場合は、その1つ手前の文字（「び」）を対象として繋ぐ。
  - **小書き文字（拗音・促音）の処理**: 「ゃ」「ゅ」「ょ」「っ」などの小文字は、大文字（「や」「ゆ」「よ」「つ」）に変換した上で判定する。
  - **重複単語の禁止**: 一度使った単語を再入力した場合は、誤った入力（ひらがな以外やしりとり不成立時と同じ）とみなし、入力文字を震えさせながら消去する。これはシステムが入力した単語、プレイヤーが入力した単語の両方に適用される。
- **しりとりルール・ゲームオーバー条件**:
  - 「ん」で終わる単語を入力した場合は、即座にゲームオーバーとなる。
  - 制限時間が0秒になった場合もゲームオーバーとなる。
- **スコアシステム**:
  - しりとりを一回成功させるごとに、スコアに100点ずつ加算。
